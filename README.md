# 🎒 RPG Inventory Manager

Um sistema de inventário para RPG desenvolvido em Java com operações CRUD via terminal.

## 🎯 Funcionalidades

- **Adicionar** itens ao inventário
- **Listar** todos os itens
- **Atualizar** dados de um item
- **Remover** itens do inventário

## 🛠️ Tecnologias

- Java (puro, sem frameworks)
- Terminal/CLI
